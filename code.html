Topic details

 

Function in JS:

Function is a block of code designed to perform a particular task. Function gets executed when it is called/ invoked.

 

Function Declaration:

  • JS function is defined with a function keyword, followed by name, followed by parentheses ( ).
  • Naming convention remains the same as the variables.
  • The code to be executed by the function is placed inside the curly braces {   } .
  • Example: function name() {

// code to be executed.

      }

Function Expression: 

  • Function expressions can be stored in a variable.
  • After the expression has been stored in a variable, the variable can be used as a function.

 

Function Parameters:

  • Function parameters are listed inside the parentheses (  ) , separated by commas , .
  • Example: function name(parameter1, parameter2)

 

Function Invocation:

  • The code inside the function will be executed when the function is invoked (called).
  • Function arguments are the values received by the function when it is invoked.
  • Function can be invoked in JavaScript or in HTML when an event occurs (like on-click of button).

 

Function Return:

  • The function will stop executing when it reaches the return statement.
  • The return value will be returned back to the caller.

 

Scope of variables in Function: 

  • Variables declared within the function are called a local variable to the function, the scope of that variable is only within the function.
  • Those variables can be accessed from within the function.
  • Outside the function, the local variable is undefined.
